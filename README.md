# Word AI 添削アプリ (Word AI Proofreader App)

Wordファイルの文章をAI（OpenAI API）を使って自動的に添削し、**変更履歴（校閲モード）**と**コメント**を付与した状態で出力するデスクトップアプリケーションです。

## 主な機能

- **AIによる文章添削**: OpenAIのモデルを使用して、自然な日本語への修正を提案します。
- **変更履歴の自動記録**: 修正箇所をWordの「変更履歴の記録」機能を使って反映するため、どこがどう変わったか一目でわかります。
- **修正理由のコメント挿入**: なぜそのように修正したのか、AIによる理由をWordのコメントとして挿入します。
- **直感的なGUI**: プログラミングの知識がなくても、APIキーとファイルを選択するだけで簡単に実行できます。

## セットアップ

### 1. 準備
- Python 3.10 以上がインストールされている必要があります。
- OpenAIのAPIキーが必要です。

### 2. 依存ライブラリのインストール
リポジトリをクローンまたはダウンロードした後、以下のコマンドを実行して必要なライブラリをインストールします。

```bash
pip install -r requirements.txt
```

## 使い方

1. アプリケーションを起動します。
   ```bash
   python main.py
   ```
2. 画面上の入力欄に **OpenAI APIキー** を入力します（環境変数 `OPENAI_API_KEY` が設定されている場合は自動で入力されます）。
3. 「参照」ボタンから、添削したい **Wordファイル (.docx)** を選択します。
4. 「添削を開始する」ボタンをクリックします。
5. 処理が完了すると、元のファイル名に `_添削済み` が付与された新しいファイルが同じディレクトリに保存されます。

## 注意事項
- 現在、`OpenAIClient` クラスはテスト用のモックモードで動作しており、一部の共通語句（「ます」→「る」など）に対してデモ用の修正を行います。実際のAPI連携を本格的に行う場合は、`openai_client.py` 内のロジックを調整してください。
- 処理対象は `.docx` 形式のファイルのみです。
